## 🔧 Tools & Techniques

| Category | Tools/Methods |
|----------|--------------|
| **Domain Recon** | Whois, DNS enumeration, subdomain brute-force, certificate transparency |
| **Email OSINT** | Hunter.io, theHarvester, breach database checks |
| **Social Media** | Username enumeration, profile correlation, Sherlock |
| **Network** | Shodan, Censys, port scanning, service fingerprinting |
| **Metadata** | ExifTool, FOCA, document analysis, GPS extraction |
| **Web** | Google Dorks, Wayback Machine, cached pages, robots.txt |
| **Geolocation** | IP geolocation, image EXIF, WiFi BSSID lookup |

## ⚖️ Legal & Ethical Notice
All tools in this repository are intended for **authorized security assessments only**. Always obtain written permission before conducting reconnaissance on targets you do not own. Unauthorized use may violate local and international laws.
